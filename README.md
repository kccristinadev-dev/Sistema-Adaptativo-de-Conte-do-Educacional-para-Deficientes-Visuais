# Sistema-Adaptativo-Educacional-para-Deficientes-Visuais
Desenvolvimento de um sistema web adaptativo com a finalidade de converter atividades pedagógicas em formatos acessíveis para estudantes com deficiência visual, em conformidade com a meta 4.5 da ODS 4 (Educação de Qualidade), a ser implementado no período estimado de 4 meses.
